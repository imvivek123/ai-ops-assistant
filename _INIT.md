# AI Operations Assistant

Production-ready multi-agent AI system for business operations automation.

## Agents

- **Meeting Summarizer**: Extracts decisions, actions, metrics from transcripts
- **Data Analyzer**: CSV analysis with statistics and insights
- **Document Generator**: Professional business documents

## Quick Start

```bash
pip install -r requirements.txt
python setup_samples.py
python main.py --task meeting --input sample_data/meeting_transcript.txt
```

See full documentation in the project files.
